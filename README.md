# Login Panel

A browser-based login page with user authentication built using HTML, JavaScript, and Bootstrap.

## Features

- Email validation on login
- Password masking
- Blocked after 3 failed attempts (with countdown timer)
- Admin page accessible only after successful login
- Login state stored using localStorage
- Safe logout functionality

## Tech Stack

- HTML5
- JavaScript
- Bootstrap

## How to run

1. Clone the repository
```bash
   git clone https://github.com/yaren-atici/login-panel.git
```
2. Open `index.html` in your browser — no server needed!

## Project Structure

| File | Description |
|------|-------------|
| `index.html` | Login page with email, password, and validation |
| `admin.html` | Admin panel, accessible only after successful login |
